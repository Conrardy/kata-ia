# Projet C #

## Prérequis

- .NET SDK (version 5.0 ou supérieure)

## Installation

1. Clonez le dépôt :

    ```bash
    git clone https://github.com/Conrardy/kata-ia.git
    cd kata-ia/csharp/Kata-IA-refactoring
    ```

2. Restaurez les dépendances :

    ```bash
    dotnet restore
    ```

3. Compilez et testez le projet :

    ```bash
    dotnet test
    ```
