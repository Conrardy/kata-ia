# Projet TypeScript

## Prérequis

- Node.js (version 14.x ou supérieure)
- npm (version 6.x ou supérieure)

## Installation

1. Clonez le dépôt :

    ```bash
    git clone https://github.com/Conrardy/kata-ia.git
    cd kata-ia/ts
    ```

2. Installez les dépendances :

    ```bash
    npm install
    ```

3. lancez les tests:

    ``` bash
    npm run test -- --watch
    ```
